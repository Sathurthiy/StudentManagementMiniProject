# StudentManagementMiniProject
It's a Console-based project. It contains features like adding a new student, updating, deleting, finding toppers and Searching a student . I have used array concepts,constructor.loops and I have created separate functions for each operations.In this i have used switch statement to enter user based choice for performing operations.There are various options like:
   * Adding Student
   * Viewing Student
   * Search student by ID
   * Delete Student
   * Update Student
   * Finding Topper
   * Exit

*Add Student
   To add Student i have created a separate function and getting the input from user like id,name and marks.In getting ID and marks there is a validation step of avoiding duplicate ID's and while getting marks, negative marks are not allowed.In adding Student if more students are added than the declared array ,a meesage will be displayed as Storage full.
*View Student
   In viewing student all student details are displayed.
*Search Student by ID
    In this first student ID is searched ,if found, student details will be displayed else no student found will be displayed.
*Delete
    In this, the ID of the student who needs to be deleted will be get from user first.Next the particular student will be deleted.
*Update Student
    first the id of the student whose details need to be updated will be get from user.Then user"s choice is asked i.e to update name only or marks only or both.Based on user choice the details are updated.
*Finding topper
     Based on the marks of all students the topper is found and all details of topper is displayed to user.
*Exit
    The project Exits.
